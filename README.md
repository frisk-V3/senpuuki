# senpuki

このプロジェクトは、Javaのオブジェクト指向の原則（カプセル化、責務の分散）に基づいて設計された扇風機シミュレーションプログラムです。

## ファイル構成
合計5つのクラスファイルで構成されています。

1.  **FanSettings.java**: 風量設定（LOW/MEDIUM/HIGH）の定数を管理します。
2.  **Motor.java**: 物理的な回転動作をシミュレートします。
3.  **ElectricFan.java**: 扇風機本体のロジック（状態管理）を担います。
4.  **RemoteControl.java**: ユーザーインターフェース（操作）を担当します。
5.  **FanSystemMain.java**: システムを起動し、各部品を接続して実行します。

## 実行方法

1. すべての `.java` ファイルを同じディレクトリに保存します。
2. ターミナルまたはコマンドプロンプトを開き、コンパイルします。
   ```bash
   javac *.java
